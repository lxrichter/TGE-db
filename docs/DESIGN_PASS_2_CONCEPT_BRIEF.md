# Design Pass 2 Concept Brief

This brief accompanies the non-production concept route:

- `/design/pass-2`

Pass 2 responds to approval of Pass 1 and focuses on moving from design
architecture into a stronger platform-wide visual system.

## Scope

Pass 2 covers:

- full-height left navigation shell
- top utility bar
- Dashboard refinement
- Markets refinement
- Analysis refinement
- signature Map Explorer direction
- typography scale
- component system direction
- mobile hierarchy validation
- chart and visualization language

## Primary Shell Decision

The platform should fully embrace a permanent left-side vertical navigation.

Primary navigation:

- Dashboard / Overview
- Projects
- Plants
- Companies
- Markets
- Analysis
- Map Explorer
- Reports
- Documents

Research Ops section:

- Research Ops
- Validation Queue
- AI Assistant

Administration section:

- Platform Admin
- Users
- Settings
- Audit / Governance

The top bar should be a utility layer only:

- global search
- notifications
- AI assistant access
- user profile
- context actions

## Research Ops Boundary

Research Ops should become a dedicated workspace, not a dashboard card.

Command Center should remain internal for Admins, Editors, and Research Ops
users only. It should not become part of subscriber-facing navigation.

## Map Explorer Direction

Map Explorer should be treated as the signature platform experience.

Design requirements:

- map dominates the page
- filters are secondary
- filters are collapsible
- future clean screenshot / presentation mode is supported
- map popups route to record, research queue, and market/country context
- mobile filters become drawer or bottom-sheet behavior

## Visual Direction

The direction should remain:

- calm
- premium
- analytical
- sector-specific
- evidence-aware
- not visually noisy
- not a generic admin dashboard

## Acceptance Signal

Pass 2 is ready to move into live implementation when:

- the sidebar feels like the permanent product shell
- the top bar no longer reads as primary navigation
- Dashboard feels executive and intelligence-first
- Markets feels like market intelligence, not country administration
- Analysis feels modular and report-capable
- Map Explorer feels dominant and signature
- mobile hierarchy is credible
- component and typography direction can be reused on operational pages
